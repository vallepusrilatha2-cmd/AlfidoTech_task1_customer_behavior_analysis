**Objective :**

The customer behavior dataset was analyzed to identify purchasing patterns, customer segments, and retention trends. Data cleaning included handling missing values, removing duplicates, and creating new features such as total spending.
RFM analysis was used to segment customers based on recency, frequency, and monetary value. The analysis identified loyal customers, high spenders, and inactive users.
